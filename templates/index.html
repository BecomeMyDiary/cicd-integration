Security Tools

เครื่องมือที่ใช้ในแต่ละขั้นตอนของ Pipeline

🔐 Secret Scan

TruffleHog

ตรวจจับ API Keys, Passwords, Tokens ที่หลุดเข้ามาใน Source Code และ Git History

Git History Credentials API Keys
🧬 SAST

Bandit

วิเคราะห์ Python Source Code แบบ Static เพื่อหาช่องโหว่ เช่น SQL Injection, Hardcoded Secrets

Python Static Analysis Code Quality
📋 SCA

Safety

ตรวจสอบ Dependencies ใน requirements.txt ว่ามี CVE (ช่องโหว่ที่รู้จัก) หรือไม่

Dependencies CVE PyPI
🐳 Container

Trivy

สแกน Docker Image หา OS-level และ Library-level Vulnerabilities ก่อน Push ขึ้น Registry

Docker Image Scan OS Vulns

Stage Tool Description Status
1 TruffleHog Secret Scanning — ตรวจจับข้อมูลลับใน Code ✅ Passed
2 Safety SCA — ตรวจสอบ Dependencies ที่มีช่องโหว่ ✅ Passed
3 Bandit SAST — วิเคราะห์ Source Code หาช่องโหว่ ✅ Passed
4 Trivy Container Scan — สแกน Docker Image 🔄 Scanning
5 Cloud Run Deploy — นำขึ้น GCP Cloud Run ⏳ Pending
